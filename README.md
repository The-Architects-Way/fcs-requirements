# FCS Website Requirements Form

An interactive multi-step questionnaire for collecting school website requirements from clients.

**Live:** [miraloyd.github.io/fcs-requirements](https://miraloyd.github.io/fcs-requirements/)

## Features

- 15-page guided questionnaire with progress tracking
- Auto-save to browser local storage (resume anytime)
- File uploads via Cloudinary (logo, crest, photos, documents)
- Mobile-responsive design
- Form submissions delivered via email notification

## Technology

- Static HTML / CSS / JavaScript (no build step)
- [Web3Forms](https://web3forms.com/) for form submission
- [Cloudinary](https://cloudinary.com/) for file hosting
- Deployed via GitHub Pages
